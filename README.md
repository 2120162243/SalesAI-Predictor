# SalesAI-Predictor

## 项目简介

基于 AI Agent 的销售预测与策略推荐系统。自动整合 CRM 数据和市场数据，生成预测报告并推荐行动策略，支持多 Agent 协作分析。

## 功能

- 数据加载与预处理
- 销售预测（回归模型）- 策略推荐（基于预测结果）
- 多 Agent 协作管理

## 使用方法

```bash
pip install -r requirements.txt
python src/main.py
```

## 目录说明

- `config/` 配置文件  
- `data/` 示例数据  
- `src/` 核心代码  
- `outputs/` 输出报告
